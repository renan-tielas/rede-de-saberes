# rede-de-saberes
Uma Rede Social Comunitária com o propósito de ser uma ferramenta Autogestionada, que facilite a comunicação, a partilha, e a organização entre diversas pessoas, grupos e redes. A princípio apresenta três eixos principais: o compartilhamento de “Sementes de Saberes” como forma de Educação Popular; a comunicação organizada como ferramenta para Autogestão; e a possibilidade de partilha, trocas e comercialização entre essas pessoas, visado a Economia Solidária.
